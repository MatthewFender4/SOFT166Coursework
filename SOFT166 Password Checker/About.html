What?

This prototype has been developed to meet the practice requirement for SOFT166. The purpose is to demonstrate the web development techniques learnt.

This simple website with its prototype password checker was created by Matthew Fender, a first year student at Plymouth University

Who?

Warning!

Whilst this basic password checker can be used with or without Hue Lights, it should be noted that this is no more than a simple prototype and it should not be used in serious or professional capacity.

This website prototype contains 4 pages
  • Home: Landing page which contains the password checker
  • About: This page which provides general information
  • Overview: Describes the Password Checker in more detail
  • Hue-Lights-Key: Shows the Hue Light patterns for any input password
