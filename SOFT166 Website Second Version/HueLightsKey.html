The Hue Lights Pattern Key

The pattern of the Hue Lights indicates the strength of the Password; this system employs one final coloured light which will be green if the password has passed all tests and red if the password has failed overall. Please note that the tests for lowerCase, upperCase, numeral and special-character are binary so need only a red hue if the character is not present and a green hue if the character is present. Password length and password overall strength use coloured lights with red, yellow and green.

To provide confidence to the user: All lights will be ON and RED initially when the system is ready for password input.

The Criteria and Results Key after password input

Light# Strength Score Colour Green. Password contains: Colour Red. Password contains:
1 +1 if present 1 or more lower case letters (success) NO lower case letters
2 +1 if present 1 or more UPPER CASE letters (success) NO UPPER CASE letters
3 +1 if present 1 or more Numerals (success) NO Numerals
4 +1 if present 1 or more Special Characters (success) NO Special Characters
5 +1 if 6 or 7 chars long; +2 if 8 or more. at least 8 Characters (success) less than 8 characters. Yellow if > 5 and less than 8.
6 Overall: -2 if common word included all criteria met (success) Strong at least one shortfall(see rows above) Weak or Very Weak

As each required element is included in the password the strength of the password improves, with length and inclusion of a common word having the most significant impact on the password strength.
